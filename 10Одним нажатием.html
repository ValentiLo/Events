Одним нажатием

Мы настроили обработчик, теперь инструкции выполняются только при нажатии клавиши ESC.

Нам осталось удалить класс modal--show у попапа, чтобы он действительно закрывался каждый раз при нажатии ESC.

Для этого будем использовать уже знакомый нам метод remove объекта classList.

  1. В обработчике клавиатурного события, внутри условия, удалите вывод в консоль и добавьте удаление класса modal--show у попапа.
  2. Закройте попап, нажав клавишу ESC. Не забудьте поставить фокус в мини-браузер.

var popup = document.querySelector('.modal');
var openPopupButton = document.querySelector('.button-open');
var closePopupButton = popup.querySelector('.button-close');

openPopupButton.addEventListener('click', function (evt) {
  evt.preventDefault();
  popup.classList.add('modal--show');
});

closePopupButton.addEventListener('click', function () {
  popup.classList.remove('modal--show');
});

document.addEventListener('keydown', function (evt) {
  if (evt.keyCode === 27) {
    popup.classList.remove('modal--show');
  }
});
